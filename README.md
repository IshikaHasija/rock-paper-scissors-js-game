# 🎮 Rock Paper Scissors Game

A simple **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript** where the player competes against the computer for 5 rounds.

## 🚀 Features
- Player vs Computer gameplay
- Random computer choice using `Math.random()`
- Scoreboard for player and computer
- 5 rounds gameplay
- Final winner announcement
- Replay button to restart the game
- Dynamic UI updates using DOM manipulation

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📚 What I Learned
- Handling click events with `addEventListener`
- Using arrays and random number generation
- Writing game logic with `if-else` conditions
- Updating the DOM dynamically
- Managing game state with variables
- Creating a replay functionality

## 🎯 How the Game Works
1. The player selects **Rock, Paper, or Scissors**.
2. The computer randomly selects one option.
3. The winner of the round is decided based on game rules.
4. Scores are updated after each round.
5. After **5 rounds**, the final winner is displayed.

## 🔄 Replay
After the game ends, the **Replay button** resets the scores and starts the game again.
